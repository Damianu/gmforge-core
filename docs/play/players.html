How do they join?

Players can join using any web browser, just as long as you have the correct network settings configured. It is recommended they use the latest version of Chrome. if they have their own copy of GM Forge, they can join via the main menu.

Local Network

Share your invitation link with them, and have them paste it into the URL bar.
The link you send should look like http://[local ip address]:30000/join

Over the Internet

Share your invitation link with them, and have them paste it into the URL bar.
The link you send should look like http://[public ip address]:35536/join unless you have configured the ports differently

Invitation Links

You can find invitation links in the game options section of the main menu (The little gear in the top right)

You can either port forward port 35536 or enable Universal plug and play
