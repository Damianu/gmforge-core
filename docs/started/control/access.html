Access Controls

Typically accessed by either right clicking on an asset, or clicking on a "lock" symbol


Individual Access

Owner : User can duplicate/delete the asset
Rights : User can make edit the asset and adjust access on the asset
Visible : User can view the asset in a Read Only Mode

*Assigning 'Visible' access on a map will reveal the entire map to a user

No Access : User can't access this asset

Default Access

GM Only : Only GMs have 'Rights' access to this asset
Players : Everybody has 'Rights' access to this asset
Custom Macro : You can use a macro to assign automatic 'Rights' access to the asset
